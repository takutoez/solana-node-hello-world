# solana-node-hello-world

元記事: https://github.com/solana-labs/example-helloworld

https://github.com/takutoez/solana-rust-hello-world のSolana programを実行するnodeクライアント

## セットアップ
```
yarn install
```

## 実行
```
node index.js
```
